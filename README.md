# Web Place - Farm E-commerce Store

Independent e-commerce storefront for farm products.

## Quick Start

```bash
npm install
npm run dev    # Runs on port 3001
```

## Architecture

This app communicates with `farm-health-app` (port 3000) via REST API for:
- Product/inventory data
- Stock deduction after payment
- Sales record sync

## Environment Variables

See `.env.local` for required configuration.
